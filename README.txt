QUIZ LIVE - FILE DEFINITIVI

1) Apri il terminale nella cartella quiz-live
2) Esegui: npm install
3) Esegui: npm start
4) Apri nel browser: http://localhost:3000/host.html

FUNZIONI INCLUSE
- stanza con codice
- QR code generato dal server
- ingresso giocatori da smartphone
- classifica live
- 4 tipi di domanda: multiple, truefalse, text, order
- punteggio con bonus velocità
- 3 modalità di visualizzazione:
  host-only        = domanda solo sullo schermo host
  host-and-player  = domanda sullo schermo e sugli smartphone
  player-only      = domanda solo sugli smartphone

FILE PRINCIPALI
- server.js
- public/host.html
- public/player.html
- public/questions.js
- package.json
